# KcpSharpN
[English version](https://github.com/new1271/kcp/blob/master/README.md) <br/>
## 簡介
KcpSharpN 是一個基於 [原始 KCP 專案](https://github.com/skywind3000/kcp) 的程式碼並手動翻譯成 C# 版本的協定實現程式庫<br/>
## 什麼是 KCP
KCP 是一個快速可靠的協定。據原作者所言，相較於TCP，它能通過浪費 10%-20% 頻寬為代價，換取平均延遲降低 30%-40%，且最大延遲降低三倍的傳輸效果。<br/><br/>
想了解更多資訊，可以查看 KCP 專案官網的[文件](https://github.com/skywind3000/kcp/wiki/Home).
